# Keyboard Video Mouse Switch

KVM switches provide remote management to access and control servers.

## HW spec

- Part Number: MPU8032DAC-35-B4-3E
- Version: 2.12.4.26064
- KVM dongle: MPUIQ-VMCHS

## Management

- serial port (SETUP)
- 1G ethernet management port
